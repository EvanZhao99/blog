# 二进制

## 符号
- 0b: 二进制`0b11===3`
- 0x: 十六进制`0x80===128===0b10000000`

## 进制转换
- 十进制-》二进制：除二取余数
- 二进制-》十进制：2的n次方
- 二进制-》十六进制：分成四位一组

## 按位操作符
- 与`&`: 两个均为1则为1
- 或`|`: 至少有一个1
- 非`~`: 取反
- 亦或`^`: 有且只有一个为1（两个相反）

## 位移
- 左移`<<`
- 有符号右移`>>`
- 无符号右移`>>>`： 如果为负数，会把符号位当成‘1’计算

## js进制转换
- toString
  ```js
    (3).toString(2) // 11
    (18).toString(8) // 22
    (18).toString(16) // 12
  ```
